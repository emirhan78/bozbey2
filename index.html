Kurulum ve kullanım

  1. Bilgisayarınızda Python ve bağımlılıkların kurulu olduğundan emin olun.
  2. Proje klasörüne gidin ve bağımlılıkları yükleyin.
  3. Uygulamayı başlatın ve arayüzü kullanmaya başlayın.
cd bozbey
pip install -r requirements.txt
python main.py

Not: Harita, raporlama ve bildirim özelliklerinin çalışması için ilgili anahtar/dosya yapılandırmalarının ( google-services.json, ServiceAccountKey.json vb.) doğru konumda olması gerekir.
